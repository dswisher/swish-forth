# Tinkering With Forth

There are two parts to this repo.

The first part was an attempt to implement Forth for the 6502, targeting the X16 Commander.
Nothing against the X16, but the limited registers on the 6502 made the implementation tricky, so I have largely abandoned this (at least for now).
For more details, see the [README](6502/README.md).

The second part is implementing Forth for RISC-V.
See the [README](riscv/README.md) for more details.
